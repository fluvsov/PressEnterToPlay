# PressEnterToPlay
#🇷🇺 Русский
PressEnterToPlay — консольная Python-игра. Игрок жмет Enter и получает число от 0 до 9999. Победа засчитывается, если выпадает комбинация из одинаковых цифр (например, 777 или 4444). Иначе — проигрыш. Игра работает в бесконечном цикле.
УправлениеEnter — Запуск / Повтор
#🇺🇸 English
PressEnterToPlay is a Python console game. Press Enter to get a number from 0 to 9999. You win if the number contains matching repetitive digits (like 777 or 4444). Otherwise, you lose. The game runs in an infinite loop.
ControlsEnter — Play / Loop again
