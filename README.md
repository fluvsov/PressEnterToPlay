# 🌸 PressEnterToPlay

A minimalist, terminal-based mini-game with vibrant neon colors.

---

## 🇷🇺 Русский
**PressEnterToPlay** — консольная Python-игра. Игрок жмет Enter и получает число от 0 до 9999. Победа засчитывается, если выпадает комбинация из одинаковых цифр (например, 7777 или 4444). Иначе — проигрыш. Игра работает в бесконечном цикле.

* **Управление:** `Enter` — Запуск / Повтор

---

## 🇬🇧 English
**PressEnterToPlay** is a Python console game. Press Enter to get a number from 0 to 9999. You win if the number consists of 4 identical digits (like 7777 or 4444). Otherwise, you lose. The game runs in an infinite loop.

* **Controls:** `Enter` — Play / Loop again

* **操作方法:** `Enter` — 実行 / 再試行
